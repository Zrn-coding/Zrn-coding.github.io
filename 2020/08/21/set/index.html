set的使用(新)
發表於2020-08-21|更新於2020-08-23|C++筆記
|字數總計:1k|閲讀時長:5分鐘|閱讀量:

set

一個紀錄是否有出現過的集合

標頭檔

構造器 && 初始化

  1. 默認構造器： empty container constructor
  2. 範圍構造器： range constructor
  3. 複製構造器： copy constructor

insert

重載函數 形参
插入單個值 pair<iterator,bool> insert (const value_type& val);
有位置提示 iterator insert (const_iterator position, const value_type& val);
範圍 template void insert (InputIterator first, InputIterator last);

輸出

erase

set刪除已存元素的方式，常見共有三種。

輸出

find

尋找set中某一個值的位置，如果找到的位置為st.end()代表尚未找到

輸出

count

判斷set中，某值是否已出現

輸出

clear

清除整個set的工具

輸出

empty

如果 set 為空，empty 返回 true;

size

返回 set 容器元素個數

multiset

可以記錄出現次數的set

標頭檔

使用

跟set的使用幾乎一模一樣，差別在count的回傳值不會再只是bool，而是出現的次數(int)

延伸比較

與priority_queue(優先佇列)相比，multiset取出任意一個元素要O(log n)**，但priority_queue要O(n)**。（這就是它叫做queue的原因）
