map的使用(新)
發表於2020-08-21|更新於2020-08-24|C++筆記
|字數總計:1.6k|閲讀時長:8分鐘|閱讀量:

map

就是一個對應表的概念，一對一的形式，跨越型態的限制

標頭檔

構造器 && 初始化

  1. 默認構造器： empty container constructor
  2. 範圍構造器： range constructor
  3. 複製構造器： copy constructor

下標操作

map最常使用的方法就只是直接使用mp[A] = B的方式互相對應

insert

重載函數 形参
插入單個值 pair<iterator,bool> insert (const value_type& val);
有位置提示 iterator insert (const_iterator position, const value_type& val);
範圍 template void insert (InputIterator first, InputIterator last);

輸出

erase

map刪除已存元素的方式，常見共有三種。

輸出

find

尋找map中某一個key值的位置，如果找到的位置為mp.end()代表尚未找到

輸出

count

判斷map中，key值是否已出現

輸出

clear

清除整個map的工具

輸出

empty

如果 map 為空，empty 返回 true;

size

返回 map 容器元素個數

multimap

就是一個對應表的概念，一對多的形式，跨越型態的限制

標頭檔

使用

跟map的操作大部分相同，但不能使用下標[]功能，插入值僅能用insert
